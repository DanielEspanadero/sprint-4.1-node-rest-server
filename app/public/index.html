🧑🏻‍💻 Entrega 4.1: Node REST Server 🧑🏻‍💻

Instruccions del projecte

Resolució del lliurament 4.1 titulat - Node REST Server - on creo una api rest de resposta ràpida. A més del que demana l'exercici he afegit front-end per aprendre a integrar-lo en un servidor d'express i entendre l'estructura de la millor manera possible, així doncs, hi ha dues maneres de comprovar aquests exercicis; la primera seria a través del navegador web i la segona, seria utilitzant postman.

🏛 Estructura 🏛

La manera que utilitzo per crear i organitzar el servidor és express basat en classes, ja que considero que és més senzill d'escalar i mantenir el codi relativament ordenat. A la carpeta models és on es troba la lògica del servidor i els controllers, les routes i els middlewares els guardo a les seves carpetes corresponents.
